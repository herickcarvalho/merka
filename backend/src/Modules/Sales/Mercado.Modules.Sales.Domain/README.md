# Sales.Domain

Camada de domínio do módulo **Sales**.

Pastas preparadas (vazias por enquanto, propositalmente — nenhuma regra
de negócio foi implementada):

- `Entities/`      → Entidades e Aggregate Roots do módulo.
- `ValueObjects/`  → Objetos de valor (ex.: dinheiro, medidas, códigos).
- `Events/`        → Domain Events específicos do módulo.
- `Repositories/`  → Interfaces de repositório (contratos), implementadas
  na Infrastructure deste mesmo módulo.
- `Exceptions/`    → Exceções de regra de negócio do módulo.
